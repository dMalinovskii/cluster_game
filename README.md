# Cluster Game

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

## Запуск тестов

```bash
npm run test
```

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`.


## Технологии

- PixiJS 6
- TypeScript
- Webpack 5
- jest
- Webpack Dev Server
